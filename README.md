## Explanation: 3D Scene Graph
A 3DSG is a semantically rich and compact representation of a scene. It is commonly represented as a hierarchical structure with multiple layers of abstraction, such as a root node, rooms, objects and object parts. Each node contains semantic information and relation
ships to other nodes in the graph. This representation is often used in robotics
and computer vision to reason about environments in a structured way.
